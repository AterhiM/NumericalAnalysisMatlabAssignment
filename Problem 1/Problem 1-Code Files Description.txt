li.m

This file contains the definition of a function called 'li()', which returns a vetor that has the evaluation of Lagrange polynomials through a given point x.
----------------------------------------------------------------

Lagrange.m

This file contains the definition of a function called 'Lagrange()', which computes 
Lagrange polynomials trough {y0, y1, ..., yM}.
These polynomials are given in a matrix form.
In the other hand, it returns Lagrange interpolation function 'pn'.
----------------------------------------------------------------

wn.m

This file contains the definition of a function called 'wn()', which computes the function 
w_{n+1} mentionned in the lecture notes.
----------------------------------------------------------------

This file contains the script to compute the upper bound for all given n.(n = 2, 4,8, 16).
